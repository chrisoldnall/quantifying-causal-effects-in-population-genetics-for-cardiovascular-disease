# PhD Thesis Repository: Proteomics MR and GWAS Analyses for Cardiovascular Disease

This repository accompanies the doctoral thesis:

**“Pleiotropy-Robust Mendelian Randomisation in Proteomic Studies of Cardiovascular Disease”**

It provides structured access to data, analysis pipelines, and supplementary materials supporting the results presented throughout the thesis.

---

## Repository Structure

- `chapter6/` — Dual-site proteomics analyses (Generation Scotland and UK Biobank)
- `chapter7/` — Pleiotropy-robust Mendelian randomisation analyses
- `chapter8/` — Genome-wide association analyses of cardiovascular outcomes

Each directory contains descriptions of analytical workflows and links to associated datasets.
